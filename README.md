・アプリケーション概要  

本好きの皆様へ、

アマゾンのレビューだけで本当に満足ですか？
もっと深くその本について議論を交わし、
理解を深めたくはないですか？

こんな方にオススメです！

・その本についての理解を深めたい方
・レビューだけでは語り足りない方
・本好きの仲間が欲しい方
・色んな人の見解を知りたい方
・普通に本が好きな方！

そんな人達のためのプラットフォームを作りました。
是非この機会にご活用ください。
